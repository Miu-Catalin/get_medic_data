# Exemplu script get medic data

## Requirements

python >= 3.6

## Instalare

Instaleaza python venv

```bash
python -m venv venv
```

Activeaza venv

linux

```bash
source venv/bin/activate
```

Instaleaza dependinte

```bash
pip install -r requirements.txt
```

## Utilizare

```bash
python get_medic_data.py
```

verifica fisierul in folderul export
